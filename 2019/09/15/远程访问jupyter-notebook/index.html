  最近在学习李航的《统计学习方法》，有大神用python代码实现了书中算法，并将其用jupyter notebook的形式开源。由于本人的笔记本和实验室台式机经常分处两地，不方便来回同步，故想到将代码全部放到服务器上，这样无论何时何地均可访问。

准备工作

  在安装jupyter notebook之前需要电脑配置好Python环境，然后推荐使用pip安装jupyter notebook。
  Python官网安装页面：https://www.python.org/downloads/release/python-374/。由于我的服务器是Windows，故选择Windows x86-64 executable installer即可，以下默认服务器为Windows。然后 pip install jupyter即可。
